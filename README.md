# 📚 Study Planner

A minimal study planning web app built with vanilla JavaScript.  
Track subjects, study hours, progress, and productivity — all stored locally in the browser.

---

## ✨ Features

- Add subjects with exam date, difficulty, and required hours
- Track studied hours (+0.5h / +1h)
- Automatic progress calculation
- Days-left countdown
- Monthly calendar view
- Productivity charts (subject distribution & weekly activity)
- LocalStorage data persistence
- Responsive Bootstrap layout

---

## 🛠 Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Day.js
- Chart.js
- FullCalendar.js
